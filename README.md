# ProjectDMGMedia

This project is created with below pattern,

* Frameworks:
  - WebdriverIO
  - Cucumber
  - Typescript
    
* Features:
  - Page Object Pattern
  - Web automation on Chrome
  - Mobile automation using Appium on Android Studio

* Reporting
  - Cucumber html report
 
    
## For Installation

Run the below command

```bash
npm install
```
